# KillStreaks - Ultimate PvP Enhancement Plugin

Transform your Minecraft server into a competitive PvP paradise with KillStreaks, the plugin that makes every player kill matter and rewards strategic combat with permanent power progression.

## What Makes KillStreaks Unique

KillStreaks isn't just another combat plugin - it's a complete PvP ecosystem that fundamentally changes how players interact:

- **Blood Essence System**: Earn magical essence from killing players, spend it on permanent upgrades
- **Legendary Weapons**: Craft powerful items with unique abilities that shape combat
- **Dynamic Bounties**: Real rewards instead of money - hunt players for gold blocks, diamonds, and epic items
- **Pure PvP Focus**: Everything revolves around player-vs-player combat (no mob grinding rewards)

## Core Features

### Blood Essence - Your Path to Power
Every player starts with Blood Essence, a magical currency earned exclusively from killing other players. Spend it on permanent upgrades that last forever:

- **+1 Heart** (100 Essence): Permanent health increase
- **Speed I** (150 Essence): Permanent movement speed boost
- **Regeneration** (200 Essence): Constant healing effect
- **Luck** (120 Essence): Better loot and rare drops

### Killstreaks - Combat Rewards
Climb the ranks with consecutive player kills:

- **2 Kills**: Strength I + Fire Resistance
- **4 Kills**: Strength II + Fire Resistance + Speed II
- **6 Kills**: Unlock Dash ability (Q to teleport short distances)
- **10+ Kills**: +5 Hearts + Legendary Killstreak Sword

### Bounties - The Hunt
When players die with valuable items, they get marked with bounties showing real rewards:

- **Low Tier**: 4 Gold Blocks, 2 Diamonds
- **Medium Tier**: 8 Gold Blocks, 4 Diamonds, Mace weapon
- **High Tier**: 16 Gold Blocks, 8 Diamonds, Phoenix Feather
- **Epic Tier**: 32 Gold Blocks, 16 Diamonds, Bounty Blade
- **Legendary Tier**: 64 Gold Blocks, 32 Diamonds, Bounty Blade + Dash Crystal

## Legendary Weapons

### Bounty Blade
**Recipe**: 3 Netherite Ingots + Stick + Blaze Powder
- Sharpness VI
- Fire Aspect II
- The ultimate sword for PvP combat

### Mace (Legendary Weapon)
**Recipe**: 5 Iron Ingots + 3 Sticks + Nether Star
- **25% Lifesteal**: Heal from damage dealt
- **Knockback**: Launch enemies away
- **Armor Piercing**: +2 damage that ignores armor
- Critical hit effects and particles

### Phoenix Feather
**Recipe**: 2 Feathers + 2 Gold Ingots + Nether Star
- Single-use revival on death
- Perfect for clutch moments

### Meteor Staff
**Recipe**: Fire Charge + 2 Blaze Powder + Blaze Rod + Nether Star
- Summon meteor showers (3 uses)
- Deals massive area damage

### Void Cleaver
**Recipe**: 3 Netherite Ingots + 2 Sticks
- Armor penetration damage
- High damage output

### Celestial Cloak
**Recipe**: 3 Diamonds + Nether Star + 3 Gold Blocks
- Double-speed elytra flight
- Press Q while wearing for powerful boost

### Dash Crystal
**Recipe**: 4 Amethyst Shards + Nether Star
- Press Q to dash short distances
- Cannot be lost on death
- Limited to 3 crafts server-wide

## Commands

### Main Command
```
/ks
```
Shows all available commands

### Blood Essence System
```
/ks essence
```
View your current Blood Essence and available upgrades

```
/ks essence health    # Buy +1 Heart (100 essence)
/ks essence speed     # Buy permanent Speed I (150 essence)
/ks essence regen     # Buy permanent Regeneration (200 essence)
/ks essence luck      # Buy permanent Luck (120 essence)
```

### Bounties & Leaderboards
```
/ks bounties
```
Opens a GUI showing current bounties with their rewards

### Crafting System
```
/ks craftitem <item>
```
Directly craft items without workbench recipes:
- `bountyblade` - Bounty Blade
- `mace` - Legendary Mace
- `phoenixfeather` - Phoenix Feather
- `meteorstaff` - Meteor Staff
- `voidcleaver` - Void Cleaver
- `celestialcloak` - Celestial Cloak
- `dashcrystal` - Dash Crystal (limited)

### Configuration
```
/ks config
```
Opens configuration GUI for personal settings

### Recipes Viewer
```
/ks recipes
```
Browse all available recipes and item information

### Admin Commands
```
/ks admin giveks <player> [amount]
```
Give killstreak kills to a player (default 1)

```
/ks admin setbounty <player> <amount>
```
Set a bounty on a player

```
/ks giveitem <player> <item>
```
Give any custom item to a player

```
/ks forceevent
```
Trigger a random server event
